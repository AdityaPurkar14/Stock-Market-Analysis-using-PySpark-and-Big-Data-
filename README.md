# Reliance Stock Market Analysis using PySpark

## Overview
This project is a Big Data analytics project developed using PySpark and Spark SQL to analyze historical stock market data. The project focuses on preprocessing financial datasets, performing trend and volatility analysis, and generating meaningful visualizations for business insights.

## Features
- Data preprocessing and cleaning
- Trend analysis on stock market data
- Volatility analysis
- Daily return calculation
- Moving average analysis
- Bullish/Bearish trend identification
- Financial data visualization using Matplotlib

## Technologies Used
- Python
- PySpark
- Spark SQL
- Pandas
- Matplotlib
- mplfinance

## Project Workflow
1. Load historical stock market dataset
2. Perform data preprocessing and cleaning
3. Analyze stock trends and volatility
4. Generate visualizations and statistical insights
5. Identify trading patterns and market behavior

## Visualizations
- Candlestick Charts
- Daily Return Analysis
- Volume Trend Analysis
- Moving Average Graphs

## Outcomes
- Extracted business insights from stock market data
- Identified bullish and bearish market trends
- Analyzed high-volume trading patterns
- Improved understanding of financial data analytics using Big Data tools
